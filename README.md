## Development

+ Clone repo.
```
$ git clone https://git.samedamci.me/samedamci/auctions
```
+ Create virtual environment.
```
# PyPy
$ virtualenv -p pypy3 .
```
```
# CPython
$ virtualenv .
```
+ Activate virtual environment.
```
$ source bin/activate
```
+ Install requirements.
```
$ pip3 install -r requirements.txt
```
+ Run development server.
```
# PyPy
$ pypy3 run.py
```
```
# CPython
$ python3 run.py
```
